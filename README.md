# pdf_add_bookmark
### pdftag
- Add catalogue for the scanned pdf
- 针对扫描版pdf添加目录
- 需要有目录文件，使用\t制表符组织目录
- 目录和扫描版页码有偏差手动设置起始页码

### pdfindex
- add catalogue for pdf wirtten by markdown
- 针对电子版生成的pdf添加目录
- 以pdf内容(如markdown)标题等级为依据

### pdf_outline
- extract the outline with corresponding pages form pdf
- 从pdf中提取出目录和页码(带有层次的)
